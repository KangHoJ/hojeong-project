시군구별 상권 분포 확인
- 시군구별로 상권분포를 확인해 보았는데 강남 > 송파 > 서초 순서로 상권이 많았음
가장 적은곳은 도봉,강북,동작구 순서였다
- 시군구별로 면적 데이터를 가지고와 series 를 만들고 이를 통해 면적별 시군구
상권 분포를 확인했는데 강남 > 강동 > 강북 > 강서 순서였다
-> 앞자리 강이 붙은 구가 상권이  많음을 알 수 있고 강북구는 면적대비 상권이 아주 많음
 

대분류명 확인
- 음식 , 소매의 상권 분포가 압도적으로 많았음 
-> 음식점을 차리려는 사람들이 많고 실제 음식점이 없청 많음
-> 따라서 음식점을 기준으로 어디에 차리면 좋을지 모색해 보려고함

- 구 안에서 대분류명을 확인해 보았는데 대부분 음식>소매>생활 서비스 순서이고
강남은 생활서비스랑 소매가 비슷했음 (강남 생활 서비스 발달)
